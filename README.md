# Number Guessing Game

This is a simple random number guessing game written in C.

## How it works
- The program generates a random number.
- The user is asked to guess the number.
- The program gives hints like "Too high" or "Too low".

## How to run
- Compile the program using a C compiler (gcc).
- Run the executable file.
- Enter numbers until you guess the correct one.
- its also show how many second hour and minute you took
